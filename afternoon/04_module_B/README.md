# 澳門生產力暨科技轉移中心 - 課程平台

## 專案介紹

這是一個專為澳門生產力暨科技轉移中心設計的現代化課程平台，特別針對手機端進行優化，讓市民能夠輕鬆瀏覽和報名課程。

### 🌟 專案特色

1. **溫馨親切的配色設計**
   - 採用橙色系作為主色調，避免常見的藍白搭配
   - 營造溫暖、友善的視覺體驗
   - 符合澳門本地市民的審美偏好

2. **優秀的手機端體驗**
   - 響應式設計，完美適配手機、平板、電腦
   - 適合單手操作的界面佈局
   - 觸控友好的按鈕大小設計

3. **智能課程推薦系統**（額外創意功能）
   - 在課程詳情頁面自動推薦相關課程
   - 根據課程類別智能匹配
   - 幫助學員發現更多感興趣的課程

4. **完整的雙語支持**
   - 中英文一鍵切換
   - 所有界面和內容均支持雙語顯示

5. **流暢的動畫效果**
   - 平滑的過渡動畫
   - 提升用戶體驗和視覺吸引力

## 優化項目

相對於原始的課程網站（https://register.cpttm.org.mo/training/sdb/），本專案進行了以下優化：

1. **手機優先設計**：原網站主要針對桌面端，本專案特別優化手機端體驗。
2. **現代化界面**：採用卡片式設計，視覺更清晰美觀。
3. **快速搜尋**：即時搜尋反饋，關鍵字高亮顯示。
4. **靈活排版**：提供卡片和列表兩種視圖模式。
5. **智能排序**：支持按日期、價格排序。
6. **課程收藏**：可收藏感興趣的課程，方便日後查看。
7. **相關推薦**：智能推薦相關課程，提升課程發現率。
8. **在線報名**：簡化報名流程，三步完成報名。
9. **溫馨配色**：使用橙色系替代傳統藍白配色。
10. **清晰層次**：通過字體大小、粗細、顏色區分信息重要性。
11. **流暢動畫**：添加過渡動畫，提升操作反饋。
12. **大按鈕設計**：適合觸控操作。
13. **清晰導航**：簡單直觀的頁面結構。
14. **錯誤提示**：友好的錯誤提示和引導。
15. **圖片懶加載**：提升頁面載入速度。
16. **分頁加載**：避免一次載入過多數據。
17. **本地存儲**：利用 localStorage 保存用戶偏好。

## 如何執行

### 方法 1：直接打開（推薦用於開發測試）

1. 直接在瀏覽器中打開 `index.html` 文件。
2. 注意：由於 CORS 限制，外部 JSON 文件可能無法加載，應用會自動使用內建的示例數據。

### 方法 2：使用本地服務器（推薦用於完整功能）

使用任何本地 HTTP 服務器，例如：

**使用 Python:**
```bash
python -m http.server 8000
```

**使用 Node.js (http-server):**
```bash
npx http-server -p 8000
```

**使用 PHP:**
```bash
php -S localhost:8000
```

然後在瀏覽器中訪問 `http://localhost:8000/index.html`。

## 為課程詳情頁面設計的額外功能介紹

### 🔗 課程評價與評分系統

#### 功能描述
在課程詳情頁面新增一個課程評價與評分系統，讓學員可以：

1. 查看其他學員的評價。
2. 為課程打分（1-5 星）。
3. 提交自己的評價。

#### 設計理念

1. **透明度**：讓學員能夠了解課程的真實質量。
2. **互動性**：提升學員參與感，讓他們能夠分享自己的學習體驗。
3. **改進建議**：通過評價反饋，幫助課程設計者改進課程內容。

#### 實現方式

- **評分功能**：
  - 使用星形圖標讓學員選擇評分。
  - 實時更新平均評分。

- **評價功能**：
  - 提供一個文本框讓學員輸入評價。
  - 評價提交後顯示在頁面上。

- **技術實現**：
  - 使用 LocalStorage 保存評價數據。
  - 評價數據以 JSON 格式存儲。

#### 用戶價值

- ✅ 幫助學員選擇適合自己的課程。
- ✅ 提升課程的透明度和信任度。
- ✅ 增加學員與平台的互動性。
