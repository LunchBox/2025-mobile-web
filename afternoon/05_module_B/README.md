# 課程列表與報名網站

## 專案介紹

這是一個課程報名網站，旨在提供使用者友善且易於存取的介面。網站包含首頁、課程詳情頁面和註冊頁面。

## 這個網站有什麼特色

*   **清晰的課程列表**：首頁以卡片形式展示所有課程，方便使用者快速瀏覽。
*   **詳細的課程資訊**：點擊課程卡片後，使用者可以檢視包含課程大綱、教師資訊和留言板的詳細頁面。
*   **簡易的註冊流程**：使用者可以在課程詳情頁面點擊「立即報名」按鈕，直接進入該課程的註冊頁面。
*   **互動式留言板**：課程詳情頁面設有留言板功能，讓使用者可以針對課程提問或發表意見。

# 優化項目

相較於原始網站，本次設計進行了以下優化：

*   **提升網站可及性 (Accessibility, A11y)**：
    *   為所有互動元件（如按鈕、連結）添加了適當的 ARIA 屬性 (Accessible Rich Internet Applications)，讓螢幕閱讀器使用者能更好地理解頁面內容。
    *   確保所有功能都可以透過鍵盤操作，方便無法使用滑鼠的使用者。
    *   調整了色彩對比，確保文字內容清晰可讀。
*   **改善使用者介面 (UI) 與使用者體驗 (UX)**：
    *   採用現代化的響應式設計 (Responsive Web Design)，讓網站在不同尺寸的裝置（如手機、平板、桌機）上都能有最佳的瀏覽體驗。
    *   簡化了導覽流程，讓使用者可以更直觀地找到所需資訊。
    *   統一了網站的視覺風格，提升了整體質感。
*   **程式碼結構優化**：
    *   將 JavaScript 程式碼模組化，分離了不同頁面的邏輯，提高了程式碼的可讀性和可維護性。
    *   優化了 HTML 結構，使其更具語意化。

# 如何執行

這是一個純前端專案，由 HTML、CSS 和 JavaScript 組成，不需要任何後端環境或複雜的建置流程。

若要繼續開發或預覽網站，請依照以下步驟操作：

1.  **下載專案**：將所有檔案下載到您的本機電腦。
2.  **開啟檔案**：直接在瀏覽器中開啟 `index.html` 檔案，即可看到網站首頁。
3.  **進行開發**：
    *   HTML 檔案 (`index.html`, `details.html`, `register.html`, `other.html`) 定義了頁面結構。
    *   CSS 檔案 (`style.css`) 負責網站的樣式。
    *   JavaScript 檔案 (`app.js`, `index-page.js`, `details-page.js`, `message-board.js`, `a11y.js`) 負責網站的互動功能。您可以直接修改這些檔案，並在瀏覽器中重新整理頁面以檢視變更。

# 為課程詳情頁面設計的額外功能介紹

在課程詳情頁面 (`details.html`)，我設計了以下額外功能：

*   **互動式留言板**：
    *   使用者可以在此處留下對課程的提問或評論。
    *   送出的留言會即時顯示在留言列表上，模擬真實的互動情境。
    *   此功能由 `message-board.js` 驅動，透過 DOM 操作實現動態新增留言。
*   **錨點連結 (Anchor Links)**：
    *   頁面頂部設有「課程大綱」和「留言板」的快速連結。
    *   點擊後，頁面會平滑滾動到對應區塊，提升了長頁面的導覽效率。

# 自己最滿意的部分

我最滿意的部分是**網站可及性 (A11y) 的全面優化**。

在這次專案中，我不僅僅是完成功能需求，更花費了許多心思去思考如何讓網站對所有使用者都友善。例如，我為所有按鈕和連結都加上了 `aria-label`，讓螢幕閱讀器可以清楚地告訴使用者這個按鈕的功能。同時，我也確保了所有互動都可以透過鍵盤的 `Tab` 鍵和 `Enter` 鍵完成，這對於行動不便的使用者來說至關重要。

我認為，一個好的網站不應該只為多數人服務，更應該考慮到少數群體的需求。透過這次實作，我對通用設計 (Universal Design) 的理念有了更深刻的理解，並為自己能打造出一個更具包容性的產品感到驕傲。
