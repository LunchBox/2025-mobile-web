# 澳門生產力暨科技轉移中心 - 課程平台

## 專案概述

這是一個完整的課程列表展示平台，實現了以下功能：

### ✅ 已實現功能

1. **課程列表頁面**
   - 精美的頁首設計（標題 + 標語 + 語言切換）
   - 兩種排版模式（卡片視圖 / 清單視圖）
   - 完整的課程信息展示
   - 響應式設計（手機/平板/電腦自適應）

2. **排序功能**
   - 按開課日期排序
   - 按價格排序

3. **搜索功能**
   - 模糊搜索（課程名 / 課程編號）
   - 關鍵詞高亮顯示
   - 清除搜索功能

4. **分頁功能**
   - 支持上一頁/下一頁切換
   - 顯示當前頁碼和總頁數
   - 支持外部 JSON 文件分頁數據

5. **中英文切換**
   - 完整的雙語支持
   - 一鍵切換界面語言

6. **報名流程**
   - 三步驟報名流程（選課程 → 填資料 → 確認）
   - 表單驗證（必填項檢查）
   - 成功提交提示

7. **課程詳情頁面**
   - 基礎頁面結構（預留後續開發）

## 技術架構

- **前端框架**: Vue.js 3
- **樣式框架**: Tailwind CSS
- **數據格式**: JSON
- **瀏覽器要求**: 現代瀏覽器（支持 ES6+）

## 快速開始

### 方法 1：直接打開（推薦用於開發測試）

1. 直接在瀏覽器中打開 `index.html` 文件
2. 注意：由於 CORS 限制，外部 JSON 文件可能無法加載，應用會自動使用內建的示例數據

### 方法 2：使用本地服務器（推薦用於完整功能）

使用任何本地 HTTP 服務器，例如：

**使用 Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**使用 Node.js (http-server):**
```bash
npx http-server -p 8000
```

**使用 PHP:**
```bash
php -S localhost:8000
```

然後在瀏覽器中訪問 `http://localhost:8000/index.html`

## 課程數據格式

課程數據存儲在 `data/` 目錄中的 JSON 文件中。格式如下：

```json
{
  "courses": [
    {
      "id": "CS001",
      "name": "課程名稱",
      "price": "MOP 2,500",
      "startDate": "2025-11-15",
      "seats": "20"
    }
  ]
}
```

### 字段說明

- `id`: 課程編號（用於圖片匹配和唯一標識）
- `name`: 課程名稱
- `price`: 學費（格式：MOP X,XXX）
- `startDate`: 開課日期（格式：YYYY-MM-DD）
- `seats`: 可報名名額

## 課程圖片

課程圖片應放置在 `images/` 目錄中，命名規則：

- 格式：`{課程編號}.jpg`
- 例如：`CS001.jpg`, `BZ001.jpg`
- 如果圖片不存在，會自動使用 `default.jpg`

### 創建默認圖片

由於無法直接創建圖片文件，您需要手動創建一個默認圖片：

1. 創建 `images/` 文件夾
2. 創建一個 800x600 的圖片命名為 `default.jpg`
3. 建議使用灰色背景加上 "課程圖片" 文字

## 功能說明

### 1. 視圖切換

點擊 "卡片" 或 "清單" 按鈕切換顯示模式：
- **卡片視圖**: 網格佈局，適合瀏覽多個課程
- **清單視圖**: 列表佈局，顯示更多信息

### 2. 排序功能

在下拉選單中選擇排序方式：
- **按開課日期排序**: 最近開課的排在前面
- **按價格排序**: 價格從低到高排列

### 3. 搜索功能

1. 在搜索框中輸入關鍵詞
2. 系統會即時過濾匹配的課程
3. 匹配的關鍵詞會以紅色高亮顯示
4. 點擊 "清除" 按鈕恢復完整列表

### 4. 報名流程

1. 點擊任意課程的 "立即報名" 按鈕
2. **步驟 1**: 確認選擇的課程
3. **步驟 2**: 填寫個人資料（必填項標記 *）
4. **步驟 3**: 確認提交的信息
5. 提交成功後顯示確認訊息

### 5. 語言切換

點擊右上角的 "中/En" 按鈕在中文和英文之間切換

## 響應式設計

平台支持三種屏幕尺寸：

- **手機** (< 640px): 
  - 卡片 1 列顯示
  - 搜索欄全寬
  - 分頁只顯示按鈕

- **平板** (640px - 1023px):
  - 卡片 2 列顯示
  - 搜索欄部分寬度

- **電腦** (≥ 1024px):
  - 卡片 3+ 列顯示
  - 完整功能佈局
  - 顯示頁碼信息

## 自定義配置

### 修改每頁顯示數量

在 `app.js` 中修改：
```javascript
itemsPerPage: 9,  // 改為您想要的數量
```

### 修改語言包

在 `app.js` 的 `lang` 對象中修改或添加翻譯：
```javascript
lang: {
    zh: { /* 中文翻譯 */ },
    en: { /* 英文翻譯 */ }
}
```

### 添加更多課程

在 `data/page1.json` 中添加課程對象，或創建新的頁面文件（如 `page2.json`）
