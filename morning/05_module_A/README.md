# 課程列表與報名網站

## 專案介紹

本專案是一個為「澳門生產力暨科技轉移中心」(CPTTM) 設計的現代化課程列表與報名網站。旨在提供一個清晰、易用且功能豐富的介面，讓使用者可以輕鬆瀏覽、搜尋及報名課程。

**主要特色：**

*   **雙語介面**: 完整支援繁體中文與英文，使用者可隨時一鍵切換。
*   **無障礙設計**: 提供高對比度模式，提升視覺障礙使用者的瀏覽體驗。
*   **響應式設計**: 自動適應桌面、平板及手機螢幕，確保在任何裝置上都有最佳排版。
*   **多視圖瀏覽**: 使用者可依個人偏好選擇「卡片」或「清單」模式來瀏覽課程。
*   **強大搜尋與排序**: 支援按課程名稱/編號進行即時搜尋，並可依開課日期或價格排序。
*   **動態分頁**: 非同步載入課程資料，切換分頁時無需重新整理頁面，提升流暢度。
*   **多步驟報名表單**: 引導式報名流程，包含即時資料驗證與最終確認步驟，降低使用者填寫錯誤的機率。
*   **互動式提示**: 使用 Toast 提示框給予使用者明確的操作回饋，例如「報名成功」。

## 優化項目

相較於原始的靜態網站設計，本專案進行了以下關鍵優化：

1.  **現代化 UI/UX**:
    *   採用 Tailwind CSS 框架，打造了更具現代感且風格一致的視覺介面。
    *   重新設計了課程卡片與列表，使其更具吸引力且資訊架構更清晰。
    *   加入了流暢的過渡動畫與互動效果（如自訂下拉選單、圖片放大等），提升了整體質感。

2.  **提升使用者體驗**:
    *   從靜態頁面轉為動態載入，使用者操作（如排序、搜尋、換頁）能即時反饋，無需等待頁面刷新。
    *   將語言和無障礙設定儲存在 `localStorage` 中，確保使用者在不同頁面間切換時，設定能夠保持一致。
    *   增加了返回頂部/底部的懸浮按鈕，方便使用者在長頁面中快速導覽。

3.  **技術架構升級**:
    *   使用 Vanilla JavaScript 進行開發，實現了前後端分離的基礎，課程資料透過 API (JSON) 載入。
    *   程式碼結構化，將不同頁面的邏輯（`main.js` 和 `app.js`）分離，易於維護與擴展。
    *   實現了元件化思維，例如可重複使用的 Toast 提示框、自訂下拉選單等。

## 如何執行

其他開發者可以依照以下步驟來設定與繼續開發此專案：

1.  **環境準備**:
    *   一個現代網頁瀏覽器（如 Chrome, Firefox, Edge）。
    *   一個本地 Web 伺服器。推薦使用 VS Code 的 **Live Server** 擴充功能。

2.  **啟動專案**:
    *   將所有專案檔案（`index.html`, `register.html`, `*.js`, `images/` 等）放置在同一個資料夾中。
    *   本專案的課程資料是透過 `http://localhost:8080/courses/page-*.json` 的路徑來非同步獲取。你需要確保本地伺服器能夠提供這些 JSON 檔案。
    *   使用 Live Server 或其他本地伺服器打開 `index.html` 即可開始瀏覽。

3.  **開發與維護**:
    *   **`index.html`**: 課程列表主頁面。
    *   **`main.js`**: 主頁面的所有 JavaScript 邏輯，包括資料獲取、渲染、搜尋、排序、國際化與無障礙功能。
    *   **`register.html`**: 課程報名頁面。
    *   **`app.js`**: 報名頁面的邏輯，包括表單驗證、多步驟導覽與狀態保存。
    *   **`details.html`**: 課程詳情頁的佔位檔案，待後續開發。
    *   **`images/`**: 存放課程圖片，圖片名稱應與課程編號對應（例如 `ATD192.png`），以利動態載入。
    *   **樣式**: 所有樣式皆透過 Tailwind CSS 的 Utility Classes 直接寫在 HTML 中，無需管理額外的 CSS 檔案。

## 為課程詳情頁面設計的額外功能介紹

`details.html` 頁面目前雖為佔位頁，但已規劃了豐富的擴充功能，使其成為一個完整的課程介紹中心：

*   **動態內容載入**: 頁面將根據 URL 中的課程代碼（如 `details.html?code=ATD192`）動態獲取並展示特定課程的詳細資料。
*   **詳細課程大綱**: 以時間軸或摺疊面板(Accordion)的形式，清晰展示課程的每個章節與學習重點。
*   **導師介紹**: 包含導師的頭像、簡介、專業背景與相關資歷。
*   **上課地點地圖**: 嵌入 Google Maps 或其他地圖服務，標示上課地點，並提供導航連結。
*   **相關課程推薦**: 在頁面底部，基於當前課程的分類或標籤，推薦其他相關課程，增加使用者黏性。
*   **一鍵報名**: 提供顯眼的「立即報名」按鈕，點擊後直接跳轉至報名頁面，並自動帶入該課程的資訊。

## 自己最滿意的部分

我最滿意的部分是 **跨頁面功能的一致性與狀態保持**。

透過將使用者的偏好設定（語言 `lang` 和高對比度模式 `highContrast`）儲存在瀏覽器的 `localStorage` 中，成功實現了無縫的使用者體驗。當使用者在首頁切換為英文模式後，點擊「報名」進入 `register.html` 時，報名頁面會自動以英文顯示；同樣地，高對比度模式也能在所有頁面中保持啟用。

這個設計雖然細微，但它極大地提升了網站的專業度和可用性，讓整個網站感覺像一個完整、貼心的應用程式，而不僅僅是幾個獨立網頁的集合。從規劃、實作到最終調試的整個過程，讓我對打造連貫的用戶旅程有了更深刻的理解。
